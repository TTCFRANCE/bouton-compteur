# Bouton compteur (Python / Tkinter)

## Description
Ce projet est une application simple réalisée en Python avec Tkinter.  
Elle permet d'incrémenter un compteur en cliquant sur un bouton.

## Objectif
J'ai réalisé ce projet pour apprendre les bases :
- de l'interface graphique avec Tkinter
- de la gestion d'événements (clic sur un bouton)
- de la manipulation de variables dynamiques en Python

## Fonctionnement
- Un compteur est affiché à l'écran
- Un bouton "Ajouter" permet d'augmenter sa valeur de 1 à chaque clic

## Technologies utilisées
- Python
- Tkinter

## Lancer le projet
1. Installer Python
2. Lancer le fichier :
```bash
python bouton_compteur_tkinter.py
